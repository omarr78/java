# Binary Search Tree (BST) Complexity

## Time Complexity

| Operation  | Average Case | Worst Case |
|------------|--------------|------------|
| Search     | O(log n)     | O(n)       |
| Insert     | O(log n)     | O(n)       |
| Delete     | O(log n)     | O(n)       |
| Traversal  | O(n)         | O(n)       |

## Space Complexity
- **Average/Worst Case**: O(n) (for storing n nodes)
